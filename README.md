# RangeMap

Mapa que obtem valores por determinada faixa

Faixa inicial - Faixa final - Chave\n
    1         -       5     - "Chave 1"\n
    6         -       10    - "Chave 2"\n
    20        -       30    - "Chave 3"\n
    
    n\
Obter chave para valor 7 -> "Chave 2"\n
Obter chave para valor 25 -> "Chave 3"\n
Obter chave para valor 12 -> null
