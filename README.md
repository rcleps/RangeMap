# RangeMap

Mapa que obtem valores por determinada faixa

Faixa inicial - Faixa final - Chave
    1         -       5     - "Chave 1"
    6         -       10    - "Chave 2"
    20        -       30    - "Chave 3"
    
    n\
Obter chave para valor 7 -> "Chave 2"
Obter chave para valor 25 -> "Chave 3"
Obter chave para valor 12 -> null
